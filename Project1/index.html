Overview

Given a single glass plate image with three stacked exposures (top-to-bottom order: B / G / R), split channels, align G and R to B using translation, and produce an RGB color image with minimal artifacts. For high-res TIFFs, use a coarse-to-fine image pyramid.

I implemented a baseline single-scale exhaustive search on a small window and then a pyramid search for large displacements. To improve color and aesthetics (required in 280A), I added automatic cropping, contrast stretching, white balance, a 3×3 color mapping, and gradient-based features for alignment.

Method

Pipeline

Alignment

  • Reference: B channel; G/R are translated to match B.
  • Metric: ZNCC computed on Sobel gradient magnitude with an interior crop (≈10%) to ignore borders.
  • Single-Scale: Exhaustive search within [−15, +15] for low-res images.
  • Pyramid: Downscale by ×0.5 to min side ≈ 300 px; search ±12 px at top, then ±5–7 px per level; propagate offsets by ×2.

Post-Processing (280A)

  • Automatic Cropping: find the largest rectangle of low channel-inconsistency using max(|R−G|,|G−B|,|R−B|) row/col projections.
  • Contrast: per-channel percentile stretch (1%–99%).
  • White Balance: white-patch @99% + gray-world refinement.
  • Better Color Mapping: a fixed 3×3 matrix mapping raw [B,G,R] to display RGB; learned offline and then fixed for all images.
